# LATTE++ compiler #
⚠️ User discresion is advised ⚠️

# Build #
```
make
```

After the build 'latc' binary will be created

# Clean #
```
make clean
```

# Run #
```
./latc <latte file>

Example
./latc example.lat
```
# External programs used #
    - bnfc - used to generate abstract syntax tree
